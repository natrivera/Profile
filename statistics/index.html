Statistics using Vanilla Javascript



Descriptive statistics describe the main features of a data set in quantitative terms. This calculator will generate certain descriptive statistics for a sample data set and produce a visual representation of the distribution.
To generate random data simply press "Generate"
or
Enter values separated by commas such as 1, 2, 4, 7, 7, 10, 2, 4, 5.
Then press "Run"
